What if you could treat events like data?

Create an auto-complete control that performs a server-side movie search as the user types.

Don't start searching until they've entered at least three characters.

Wait a quarter second between each keypress.

As the user types, old search requests need to be cancelled.

We will do all this with a single line of RxJS.

At the heart of RxJS is the Observable.

It has three main aspects:

1. Static Factory Methods

2. The subscribe method

3. The pipe method.
